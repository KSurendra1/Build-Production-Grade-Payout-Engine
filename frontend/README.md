# Frontend (React + Tailwind)

Merchant dashboard for Playto Payout Engine.

## Features
- Shows available and held balances
- Shows payout history with live polling
- Shows recent credit and debit ledger entries
- Allows merchant creation with initial seed credit
- Allows payout requests with UUID idempotency keys

## Dev
- `npm install`
- `npm run dev`

## Build
- `npm run build`
